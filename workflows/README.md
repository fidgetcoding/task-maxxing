# Workflows

n8n workflow JSON exports. Import these into your n8n instance to enable task sync.

- `W1-obsidian-git-task-sync.json` — syncs Obsidian task changes into Notion via git
- `W2-morgen-task-completion-sync.json` — mirrors Morgen task completion into Obsidian/Notion
- `W3-notion-done-to-obsidian-sync.json` — mirrors Notion "Done" status back to Obsidian
